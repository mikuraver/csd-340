    5 Things I've Learned Since the Course Started

  • Classes can be used to format different sections of the site separately.
  • You can put all of the styling elements for a webpage in a separate .css file and link it.
  • The dots on unordered lists can be changed.
  • You can view the source code for any website.
  • Website designs are stored in directories and can be created with only a text document, no special programs needed.
